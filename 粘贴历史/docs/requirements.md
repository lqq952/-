# 产品需求文档 — 历史剪贴板

> 版本：v1.0 | 日期：2026-06-30 | 状态：开发中

---

## 1. 项目概述

一款运行在 Windows 上的历史剪贴板管理软件。后台常驻系统托盘，自动记录用户复制的文字和图片，以卡片形式展示，支持搜索、置顶、删除与再次粘贴。

## 2. 目标用户

- Windows 电脑用户
- 经常需要回溯复制历史的人群
- 不需要技术背景，开箱即用

## 3. 功能清单

### F1 剪贴板监听
| 项目 | 说明 |
|------|------|
| 描述 | 自动监听系统剪贴板变化 |
| 支持类型 | 文字（text）和图片（image） |
| 检测方式 | 每 500ms 轮询，哈希去重避免重复 |
| 触发时机 | 软件启动即开始，后台持续运行 |

### F2 历史卡片列表
| 项目 | 说明 |
|------|------|
| 排列方式 | 按复制时间降序（最新在最上面） |
| 显示内容 | 文字显示前 200 字预览；图片显示缩略图 |
| 交互 | 点击卡片→复制到剪贴板，显示"已复制"提示 |
| 数量上限 | 最多 500 条（超出自动删最旧非置顶） |

### F3 置顶
| 项目 | 说明 |
|------|------|
| 操作 | 每张卡片有 📌 置顶按钮 |
| 行为 | 置顶项始终在列表最顶部 |
| 保护 | 置顶项不受过期清理影响 |

### F4 删除
| 项目 | 说明 |
|------|------|
| 操作 | 每张卡片有 🗑️ 删除按钮 |
| 行为 | 立即从列表和数据库中移除 |

### F5 搜索
| 项目 | 说明 |
|------|------|
| 位置 | 页面顶部搜索框 |
| 方式 | 输入即搜，对文字内容模糊匹配 |
| 作用范围 | 仅过滤文字记录，图片不受影响 |

### F6 存储时长
| 选项 | 说明 |
|------|------|
| 1天 | 24 小时后自动清理 |
| 3天（默认） | 72 小时后自动清理 |
| 5天 | 120 小时后自动清理 |

### F7 系统托盘
| 项目 | 说明 |
|------|------|
| 图标 | 显示在任务栏右下角系统托盘区 |
| 右键菜单 | 显示主窗口 / 退出 |
| 关闭行为 | 关闭窗口 = 隐藏到托盘（不退出） |

### F8 开机自启
- 默认开启
- 设置面板中可切换

### F9 全局快捷键
- `Ctrl + Shift + V` 显示/隐藏主窗口

## 4. 非功能需求

| 类别 | 要求 |
|------|------|
| UI 风格 | 简洁直观，淡蓝色主色调 |
| 图片质量 | 保留原图，不压缩 |
| 性能 | 列表滚动流畅（虚拟滚动），搜索即时响应 |
| 安装 | 提供 .exe 安装包，双击即可安装 |
| 系统 | Windows 10+ |
